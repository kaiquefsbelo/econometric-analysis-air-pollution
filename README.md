# econometric-analysis-air-pollution
# Air Pollution and Public Health in São Paulo: An Econometric Analysis

## Overview

This project investigates the relationship between air pollution, weather conditions and mortality in the city of São Paulo using real-world environmental and public health data.

The analyses were developed in **R** using data from the **Projeto PASSO (IME-USP)**.

## Research Questions

- Does air pollution affect mortality?
- Which age groups are most affected?
- How do weather conditions relate to respiratory deaths?
- Which monitoring stations present the highest CO concentrations?
- Is there a relationship between carbon monoxide and other pollutants?

## Dataset

Source:

- Projeto PASSO – IME/USP

The dataset contains daily observations including:

- Air pollution measurements
- Meteorological variables
- Daily deaths by cause
- Daily deaths by age group

## Methods

The project includes:

- Exploratory Data Analysis (EDA)
- Boxplots
- Scatter plots
- Correlation analysis
- Linear regression
- Missing data analysis

## Main Findings

• Air pollution showed a positive association with daily mortality.

• Lower temperatures were associated with higher respiratory mortality.

• Congonhas presented the highest CO concentrations among monitoring stations.

• PM10 exhibited the strongest correlation with CO.

• Individuals over 65 years old showed substantially higher cardiac mortality.
